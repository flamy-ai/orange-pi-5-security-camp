<h1 align=center>OS Installation</h1>

<br>

<h2 align=center>Download</h2>

1. Download [Ubuntu Server](https://github.com/Joshua-Riek/ubuntu-rockchip/releases/download/v1.33/ubuntu-22.04.3-preinstalled-server-arm64-orangepi-5.img.xz) for your Orange Pi 5.
    1. This link is from [this page](https://joshua-riek.github.io/ubuntu-rockchip-download/boards/orangepi-5.html).
2. Download and Install [balenaEtcher](https://etcher.balena.io/#download-etcher).
    1. If you're on Windows click [here](https://github.com/balena-io/etcher/releases/download/v1.18.11/balenaEtcher-Setup-1.18.11.exe).
    2. Make sure to run the file you downloaded to install balenaEtcher.

<br><br><br>

<h2 align=center>Write OS to MicroSD Card</h2>

1. Connect your MicroSD card to your computer.
    1. Stick the MicroSD card into an adapter and plug the adapter into your computer.
2. Run balenaEtcher.

   <img width=40% src="/src/img/balenaetcher-main-screen.png">
3. Select the Ubuntu Server OS file you downloaded.

   <img width=40% src="/src/img/balenaetcher-select-ubuntu-os-file.png">
4. Select the MicroSD card.

   <img width=40% src="/src/img/balenaetcher-select-microsd.png">
5. Hit **Flash**.
    1. Be sure you are flashing the correct file to the correct drive (the MicroSD card). Hit **Yes, I'm sure** if the details are correct.
        1. If you are uncertain, ask for help.
<br><br>   
   <img width=40% src="/src/img/balenaetcher-main-screen-flash.png">
6. When it asks **Do you want to allow this app to make changes to your computer?** and shows **Windows Command Processor**, click **Yes**.
    1. After you hit **Yes** it will begin flashing the MicroSD card, go ahead and move on to the next part while you wait.
<br><br>
   <img width=40% src="/src/img/balenaetcher-uac-prompt.png">
